- Severity rating (HIGH > MEDIUM)
- Whether the signal is sustained (structural) vs a single blip
- Cross-signal compounding: a LoB with multiple simultaneous signals (e.g. S1 + S2) should be elevated
- Anomalies that compound a signal should be noted but not replace a core signal

OUTPUT FORMAT — respond ONLY with a valid JSON object, no markdown, no preamble:
{
  "top_concerns": [
    {
      "rank": 1,
      "signal_id": "S1|S2|S3",
      "lob": "<line of business>",

    },
    ...
  ],

    "lob": "<line of business>",
    "signal_name": "<signal name>",
    "one_line_rationale": "<direct, with actual £ figures and health verdict>",
    "severity": "OPPORTUNITY",
    "impact_score": <float>
  },
  "analyst_notes": "<2-3 sentences: any cross-signal patterns, compounding risks, or additional context the CUO narrative writer should know>"

RULES:
- Use actual numbers from the payload (£ values, percentages, ratios)
- Do not hedge or soften — be direct
- Do not create findings not present in the payload
- The one_line_rationale must be a single sentence, max 30 words
- analyst_notes should surface non-obvious cross-LoB patterns
